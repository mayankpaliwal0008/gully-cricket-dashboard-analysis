## 📌 About Project
This project is a **C++ based Gully Cricket Dashboard Analysis System** designed for small-level cricket matches.  
It helps in managing match data, analyzing player performance, and storing records efficiently.

The system provides a simple console-based interface to view match statistics and player details in an organized way.

---

## 🚀 Features
- 📊 Player performance tracking  
- 🏏 Match statistics analysis  
- 💾 Data storage using file handling  
- 🖥️ Console-based dashboard  
- 📈 Easy data visualization through output  

---

## 🛠️ Tech Stack
- C++
- File Handling
- Object-Oriented Programming (OOP)
